### Project Title
Explore US Bikeshare Data

### Description
This project use python to explore data related to bike share systems for three major cities in the United Statesâ€”Chicago, New York City, and Washington. The project include code to import the data and answer interesting questions about it by computing descriptive statistics. It takes in raw input to create an interactive experience in the terminal to present these statistics.

### Files used
chicago.csv
new_york_city.csv
washington.csv

### Software used
Python 3, NumPy, and pandas installed using Anaconda
Text editor, like Sublime or Atom.
Terminal application (git bash)

### Interacitve Answers to User Input
What is the most popular month for start time?
What is the most popular day of week (Monday, Tuesday, etc.) for start time?
What is the most popular hour of day for start time?
What is the total trip duration and average trip duration?
What is the most popular start station and most popular end station?
What is the most popular trip?
What are the counts of each user type?
What are the counts of gender?
What are the earliest (i.e. oldest person), most recent (i.e. youngest person), and most popular birth years?
